# Estado da Nação — Meu Hub 3.0

## Objetivos Alcançados
Transformamos o Hub em uma central de comando dinâmica que prioriza a continuidade operacional sobre a consulta passiva.

## 1. Orquestração de Prioridade Operacional
O Hub agora reordena automaticamente seus blocos baseando-se na urgência:
- **Prioridade Máxima: Missão Ativa**. Se houver uma missão, ela domina o topo com progresso e CTA de retomada.
- **Prioridade Alta: Fila Offline**. Alerta crítico para sincronização de dados pendentes.
- **Prioridade Média: Impacto Territorial**. Novo card que mostra envios totais e missões concluídas, reforçando o valor do coletor no recorte.
- **Sugestão Contextual**: Volta para a última cidade visitada com CTAs adaptados ao status (Iniciante vs Senior).

## 2. Status de Utilidade Contextual
O "Próximo Passo" sugerido no card de status agora é inteligente:
- Se você tem uma missão iniciada, o Hub te diz para terminá-la para subir de nível.
- Se você tem pendências offline, o foco é a sincronização.
- Se está livre, o foco é fechar lacunas críticas.

## 3. Telemetria de Retorno Diário
- Implementamos `hub_retention_view` para medir a frequência com que o Hub é o ponto de entrada.
- Rastreamento de `hub_continuity_action` para entender quais gatilhos de continuidade (Missão, Fila ou Recorte) são mais eficazes.
